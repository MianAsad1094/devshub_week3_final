# DevelopersHub Internship - Final Project (Week 3)

## 📱 Project Overview
This is the final culmination for the 3rd week Flutter Development Internship at DevelopersHub Corporation. It is a fully functional, offline-first Task Management Application that combines UI/UX best practices, state management, and persistent local storage.

## ✨ Key Features
* **Custom Splash Screen:** A welcoming 3-second splash screen on app startup.
* **CRUD Operations:** Easily Create, Read, Update (Mark as Complete), and Delete tasks.
* **JSON Data Encoding:** Complex state (task title and completion status) is managed using Dart's `dart:convert` for robust JSON encoding/decoding.
* **Persistent Storage:** Integrated with `SharedPreferences` to ensure user data remains intact even after the app is completely closed.
* **Polished UI/UX:** Clean, intuitive interface with Custom App Bar, Floating Action Buttons, Dialog boxes for input, and visual feedback (Strikethrough) for completed tasks.

## 🛠️ Technologies Used
* Framework: Flutter
* Language: Dart
* Local Storage: `shared_preferences`

## 🚀 Setup Instructions
1. Clone the repository: `git clone https://github.com/MianAsad1094/devshub_week3_final.git`
2. Install dependencies: `flutter pub get`
3. Run the application: `flutter run`
